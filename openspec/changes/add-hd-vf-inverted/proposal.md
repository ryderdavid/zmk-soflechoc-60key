## Why

Create hd-Vf^ — an inverted variant of Vibranium F where top and bottom alpha rows are swapped. Same home row, same combos (with updated positions). For users who prefer curling over extending fingers.

## What Changes

- Add layer 11: hd-Vf^ (display "hd-Vf^")
- `#define HD_VF_INV 11`
- Top row gets VF's bottom row content, bottom row gets VF's top row content
- MOUSE layer-tap stays at pos 41 (swap the tap key only)
- Update cycle: QWERTY → VF → **VF^** → Pm → Au+ → Enth-M → QWERTY
- Duplicate VF's combos with updated positions for VF^

## Capabilities

### New Capabilities
- `hd-vf-inverted-layer`: VF with top/bottom rows swapped

## Impact

- New layer added, cycle becomes 6 stops
- VF combos duplicated with row-swapped positions
